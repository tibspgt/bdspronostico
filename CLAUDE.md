# Projet — App Pronostics Coupe du Monde 2026

## C'est quoi
Une application web pour un groupe d'amis (~10 personnes) 
pour faire des pronostics sur les matchs de la Coupe du Monde 2026.

## Fonctionnalités principales
- Chaque utilisateur entre son pronostic avant chaque match
- Résultats des matchs en direct via une API foot
- Système de scoring automatique basé sur la proximité 
  du pronostic avec le vrai score
- Classement général mis à jour en temps réel

## Système de points (exemple)
- Score exact trouvé → 3 points
- Bon vainqueur + bon écart de buts → 2 points
- Bon vainqueur seulement → 1 point
- Mauvais résultat → 0 point

## Utilisateurs
- ~10 amis, pas besoin de scaler énorme
- Connexion simple (email ou Google)
